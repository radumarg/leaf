### Leaf basic types

quantum computing specific types:
```leaf
bit, qubit
```

angle types: 32-bit or 64-bit floating-point values in the range [0, 2π)
```leaf
angle32, angle64
```

param type:
```leaf
let theta: param = Param("theta");
```

signed integer types:
```leaf
i8, i16, i32, i64, i128
```

unsigned integer types:
```leaf
u8, u16, u32, u64, u128
```

floating-point types:
```leaf
f32, f64
```

boolean type:
```leaf
bool
```

unit type:
```leaf
let () = ();
```
